# Objetivo:
Este repositório tem como objetivo reunir exercícios e exemplos práticos para estudo e prática de desenvolvimento web e lógica de programação.

- Exercícios de **lógica de programação**;
- Possíveis exemplos em **JavaScript** para interação e dinamismo.
